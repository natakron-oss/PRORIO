ผู้พัฒนาระบบ — ลงพื้นที่ร่วมกับผู้ใช้งานจริง

พัฒนาระบบผังเมืองดิจิทัลร่วมกับผู้ใช้งานจริงในพื้นที่ ทำให้เข้าใจสภาพปัญหา กระบวนการทำงาน และความต้องการที่แท้จริงของผู้ใช้ ไม่ใช่เพียงการคาดเดาหรือออกแบบบนกระดาษ นำข้อมูลและข้อเสนอแนะจากการใช้งานจริงมาวิเคราะห์และปรับปรุงระบบอย่างต่อเนื่อง เพื่อให้ฟังก์ชันต่าง ๆ สอดคล้องกับการทำงานของผู้ใช้มากที่สุด

ปัจจุบันมีแนวคิดต่อยอดรูปแบบการพัฒนานี้ไปสู่การขยายตลาดสู่ อบต. และหน่วยงานท้องถิ่นอื่น ๆ ที่มีลักษณะปัญหาใกล้เคียงกัน

03 — เป้าหมาย

Statement of Purpose

03 / 04
“ระบบที่ดีต้องไม่ใช่แค่ทำงานได้ แต่ต้องใช้งานง่าย เข้าใจผู้ใช้ และตอบโจทย์ความต้องการที่แท้จริง”

ข้าพเจ้ามีความประสงค์จะศึกษาต่อในสายงานที่เกี่ยวข้องกับการพัฒนาเว็บแอปพลิเคชันด้าน Frontend และการออกแบบประสบการณ์ผู้ใช้ (UX/UI Design) เนื่องจากสนใจด้านเทคโนโลยีดิจิทัล และตระหนักถึงความสำคัญของการออกแบบระบบที่ไม่เพียงทำงานได้ดี แต่ยังต้องใช้งานง่าย เข้าใจผู้ใช้ และตอบโจทย์ความต้องการของผู้ใช้งานจริง

จากประสบการณ์ที่ผ่านมา ได้มีโอกาสฝึกฝนการพัฒนาเว็บไซต์เบื้องต้นด้วย HTML, CSS, JavaScript รวมถึง React ทำให้พบว่าตนเองมีความสนใจเป็นพิเศษในส่วนของ Frontend ซึ่งเป็นจุดเชื่อมต่อระหว่างผู้ใช้กับระบบ และรู้สึกสนุกกับการแปลงไอเดียหรือแบบร่างให้กลายเป็นหน้าเว็บไซต์ที่โต้ตอบได้จริง

อย่างไรก็ตาม ยังตระหนักว่าความรู้ด้าน UX/UI Design เช่น User Research, Wireframe, Prototype และ Usability Testing ยังต้องพัฒนาต่อในเชิงลึกมากขึ้น เป้าหมายในอนาคตคือการเป็น Frontend Developer ที่ออกแบบและพัฒนาระบบคุณภาพได้ ทั้งในด้านเทคนิคและประสบการณ์ผู้ใช้ พร้อมมีส่วนร่วมในการพัฒนาผลิตภัณฑ์ดิจิทัลที่เป็นประโยชน์ต่อสังคม
